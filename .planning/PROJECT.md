# Project: BalanceWave Flutter

## Overview
BalanceWave is a mobile balance calculator for Pakistani telecom users. It allows users to calculate their net balance after tax (forward calculation) or determine the required recharge amount to achieve a desired balance (reverse calculation).

## Goals
- Convert existing React web app to a high-quality Flutter Android application.
- Maintain exact mathematical formulas and logic from the React version.
- Implement a modern, premium UI with glassmorphism and 3D animations.
- Support all major Pakistani carriers: Jazz, Zong, Telenor, Ufone, Onic, and SCOm.

## Tech Stack
- **Framework**: Flutter
- **State Management**: Provider
- **UI/UX**: Glassmorphism, Custom Animations (Animate Do)
- **Typography**: Google Fonts (Inter/Outfit)
- **Icons**: Lucide Icons (flutter_lucide)

## Target Platform
- Android (Primary)
- iOS (Secondary)
